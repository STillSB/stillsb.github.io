第一篇博客

2021-07-04 ::

咋才开始写博客？⌗

2014年12月大三下半学期跑出来实习，掰指头算算，工作已经6年半了。

我的天，时间过都好快！

心里一直也知道写博客的好处，奈何没有找到一个优雅的方式去写。总觉得博客应该是长时间维护下去的一个东西，而一想到要长时间维护下去，就异常谨慎，导致一直停滞在想法阶段，没有落实。😂

其实中途也做过几次尝试的，一开始准备在 CSDN上写来着，可能是门槛比较低，里面文章质量良莠不齐显得整个社区乱糟糟的，就没写。后来看到 博客园不错，里面的文章质量也高，心想有一天我搞个自己的个人博客多好，就又没写

好吧。。。哪有那么多理由，其实就是懒

博客与记笔⌗

平时工作，也是零零星星在记些东西的，但毕竟属于个人笔记，总会偷懒，也写的不详细。

一开始笔记都是在 印象笔记上记着的。那时还是在上大学，刚刚接触到云笔记这个概念，全局的关键词搜索与各设备同步，让我如获至宝，我在印象笔记中记了蛮多东西的，什么都喜欢往里塞，学习笔记、收藏文章、甚至各种密码，一搜就出来，很方便。后来它们对免费用户的限制越来越多，甚至对登陆设备数都限制，导致我很长一段时间都是在用他们的Web端（因为使用客户端会占用设备数）😨

后来，免费用户的体验越来越糟糕，让我不得不开始考虑迁移。比较了市面上的各种笔记后，我选了 有道云笔记。当时迁移的过程也非常艰辛，因为年代久远，已经不记得具体遇到什么困难了，总之就是格式的不兼容，很多困难就像是印象笔记为了留住用户防止他们迁移而强行设置的。因为迁移工作量巨大，我放弃了很多当时的学习笔记，毕竟那些知识随便搜搜就能在网上找到，而且我也几乎不会再用到，它们现在依然安静的躺在我的印象笔记里 😄

今年， notion免费了。当试用了notion后，我义无反顾的投入了notion的怀抱。哈哈，不过这是后话。有道云笔记是好东西，分文件夹管理和组织内容很方便，使用起来也非常符合国人的习惯，速度也快，如果有家人朋友需要使用云笔记，我会推荐有道云笔记给他们

个人笔记和博客都能记录与积累，但针对的用户不同。个人笔记更加私人，用户是自己，而博客面向的假定用户是大众。

是时候搞个属于我的个人博客了

静态博客？动态博客？⌗

我需要一个怎样的博客呢？我想了想我的核心需求：

  • 支持markdown
  • 支持代码高亮
  • 支持图片与附件
  • 简单、轻量、易部署

其实需求就这么简单，我只需要一个小巧的静态站点，让我舒适的写作。只是写几个字，请不要让我架nginx、维护数据库这一套。所以我上来过滤掉了 wordpress

hexo和 hugo这两个静态博客，都比较优秀，感觉上hugo的开发者好像响应更迅速一些，于是就选了hugo
